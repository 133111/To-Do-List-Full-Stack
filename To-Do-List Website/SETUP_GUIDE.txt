🐛 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

PROBLEM: "Username tidak ditemukan"
SOLUSI:
  1. Pastikan setup_accounts.sql sudah dijalankan
  2. Di phpMyAdmin, tabel users, cek ada data admin?
  3. Kalau kosong, jalankan setup_accounts.sql lagi

PROBLEM: "Database Connection Failed"
SOLUSI:
  1. Pastikan MySQL service running (Services > MySQL80 > Running)
  2. Di Windows, tekan: Services (win+r → services.msc)
  3. Cari "MySQL80" → klik kanan → Start
  4. Tunggu sampai status: Running
  5. Refresh browser

PROBLEM: "Token required" di API
SOLUSI:
  1. Pastikan sudah login terlebih dahulu
  2. Cek browser console (F12 → Console)
  3. Pastikan token tersimpan di localStorage
  4. Coba logout → login lagi

PROBLEM: "Data tidak tersimpan"
SOLUSI:
  1. Di API_TESTING.html, test "Add Todo" atau "Add Log"
  2. Lihat response JSON - ada error?
  3. Cek di phpMyAdmin tabel todos/activity_logs - ada data?
  4. Kalau mau, share error message-nya
