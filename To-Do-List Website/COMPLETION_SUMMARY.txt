## 🚀 QUICK START (5-10 MINUTES)

### Step 1: Backup Database (Recommended)
```
1. Open http://localhost/phpmyadmin
2. Select database "todo_list"
3. Click "Export" & download backup
```

### Step 2: Update Database
```
Option A (Recommended):
1. Open http://localhost/phpmyadmin
2. Click "Import"
3. Upload db_setup_updated.sql
4. Click "Go"

Option B (Existing Database):
1. Read MIGRATION_GUIDE.txt
2. Run migration queries di SQL tab
```

### Step 3: Test
```
1. Open index.html
2. Login
3. Click "+ Tambah Kategori"
4. Create test category & add tasks
5. Test edit, delete, etc
```

### Step 4: Learn (Optional)
```
Read PANDUAN_KATEGORI.txt untuk belajar semua fitur
```
