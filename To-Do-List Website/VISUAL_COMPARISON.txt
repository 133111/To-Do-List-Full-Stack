AFTER (New Design)
════════════════════════════════════════════════════════════════

🎨 COLOR SCHEME:
   Background: Dark navy gradient (#0f172a → #1e293b → #334155)
   Sidebar: Gradient dark (#1e293b → #0f172a)
   Buttons: Gradient indigo→darker indigo
   Cards: White dengan gradient overlay
   Text: Dark on light cards, light on dark background
   
   Color Palette:
   ├─ Primary: Indigo (#6366f1)
   ├─ Secondary: Pink (#ec4899)
   ├─ Success: Emerald (#10b981)
   ├─ Warning: Amber (#f59e0b)
   ├─ Danger: Red (#ef4444)
   └─ Dark: Navy (#0f172a - #334155)

📐 TYPOGRAPHY:
   Font: Poppins (primary) + Quicksand (headers)
   Multiple font weights (300-800)
   Clear visual hierarchy
   Better letter spacing
   Professional appearance

🎯 BUTTONS:
   Style: Gradient buttons
   Hover: translateY(-2px) + shadow glow
   Rounded: 12px (more modern)
   Shadow: Colored tints matching button
   Box-shadow: 0 4px 15px rgba(99,102,241,0.3)

💬 CARDS/SEGMENTS:
   Background: White dengan gradient overlay
   Rounded: 18px (softer)
   Shadow: 0 8px 32px rgba(0,0,0,0.15)
   Hover: Shadow increase + gradient border reveal
   Border: 1px rgba(255,255,255,0.5)
   Effects: Backdrop blur, gradient top border

🎪 ANIMATIONS:
   Smooth cubic-bezier(0.4,0,0.2,1) easing
   Fade in/slide in effects
   Smooth transitions (0.3s)
   Gradient shimmer on sidebar
   Professional feel

📱 MOBILE:
   3 breakpoints: 1024px, 768px, 480px
   Sidebar becomes responsive drawer
   Better touch targets
   Optimized spacing for small screens
