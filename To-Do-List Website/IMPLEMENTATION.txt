11. TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Page is blank
Fix: 
   - Check file is in correct folder
   - Use modern browser (Chrome/Firefox/Edge)
   - Check browser console for errors (F12)

Problem: Data lost after restart
Fix:
   - This is normal (Local Storage limitation)
   - For persistent data, upgrade to backend server
   - Or don't clear browser cache

Problem: Can't login
Fix:
   - Check username & password
   - Clear browser cache & try again
   - Register new account

Problem: Tasks not showing
Fix:
   - Refresh page (F5)
   - Check Local Storage (DevTools → Application)
   - Clear cache & re-login
