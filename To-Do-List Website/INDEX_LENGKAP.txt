## 📄 DETAIL SETIAP FILE

### 1. index.html ✏️
**Status:** DIMODIFIKASI  
**Lines Added:** ~950 lines  
**Type:** HTML + CSS + JavaScript  

**Apa yang ditambah:**
- Modal dialog untuk add/edit kategori
- CSS styling untuk modal, buttons, kategori sections
- ~500 lines JavaScript untuk:
  - `loadCategories()` - Fetch kategori dari API
  - `renderCategories()` - Render ke DOM
  - `openAddCategoryModal()` - Buka modal tambah
  - `openEditCategoryModal()` - Buka modal edit
  - `closeCategoryModal()` - Tutup modal
  - `saveCategory()` - Save kategori (create/update)
  - `deleteCategory()` - Delete kategori
  - `loadTasksForCategory()` - Load task per kategori
  - `addTask()` - Updated untuk category_id
  - `updateLogSegmentOptions()` - Update dropdown

**Key Changes:**
- Static segments (Pelajaran, Pemrograman, etc) → Dynamic container
- Category header dengan tombol "Tambah Kategori"
- Empty state jika belum ada kategori
- Modal dengan form untuk kategori baru

---

### 2. api.php ✏️
**Status:** DIMODIFIKASI  
**Lines Added:** ~150 lines  
**Type:** PHP Backend  

**Apa yang ditambah:**
- 4 action handlers baru:
  - `add_category` - POST endpoint
  - `get_categories` - GET endpoint
  - `update_category` - POST endpoint
  - `delete_category` - POST endpoint

- 4 fungsi PHP baru:
  - `addCategory($conn, $input, $userId)` - Create kategori
  - `getCategories($conn, $userId)` - Fetch all kategori
  - `updateCategory($conn, $input, $userId)` - Edit kategori
  - `deleteCategory($conn, $input, $userId)` - Delete kategori

- Modified functions:
  - `addTodo()` - Changed from 'category' (string) to 'category_id' (int)
  - `getTodos()` - Added filtering by category_id

**Key Changes:**
- New database queries untuk kategori management
- Validation untuk duplicate category names
- Foreign key management untuk task deletion
