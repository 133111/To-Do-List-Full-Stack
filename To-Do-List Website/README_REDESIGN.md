NEXT STEPS (OPSIONAL)
════════════════════════════════════════════════════════════════

Jika ingin menambahkan image backgrounds:

1. Letakkan image di folder /gambar
2. Buka index.html
3. Cari bagian CSS untuk .main-content
4. Tambahkan: background-image: url('gambar/nama-file.png');
5. Adjust opacity jika diperlukan

Example:
.main-content {
    background: url('gambar/background.png');
    background-size: cover;
    background-attachment: fixed;
}
