## 🚀 QUICK START

### 1. Update Database
**Option A - Recommended** (Fresh Start)
- Backup database Anda terlebih dahulu jika ada data penting
- Import file `db_setup_updated.sql` ke phpMyAdmin
- Atau jalankan query di file `MIGRATION_GUIDE.txt`

**Option B - Manual** (Preserve Data)
```sql
CREATE TABLE IF NOT EXISTS categories (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    name VARCHAR(255) NOT NULL,
    emoji VARCHAR(10) DEFAULT '📝',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    UNIQUE KEY unique_category_per_user (user_id, name)
);

ALTER TABLE todos ADD COLUMN category_id INT AFTER user_id;
ALTER TABLE todos ADD FOREIGN KEY (category_id) REFERENCES categories(id) ON DELETE CASCADE;
CREATE INDEX idx_user_categories ON categories(user_id);
CREATE INDEX idx_category_todos ON todos(category_id);
```

### 2. Test Fitur Baru
1. Buka `index.html` dan login
2. Lihat tombol **"+ Tambah Kategori"** di atas
3. Klik dan buat kategori pertama Anda!

## 📞 TROUBLESHOOTING

### Problem: Kategori tidak muncul
**Solution:**
- Database belum di-update
- Buka DevTools (F12) → Console
- Check untuk error messages
- Refresh halaman

### Problem: Tidak bisa simpan kategori
**Solution:**
- Check apakah sudah login
- Check server logs di backend
- Pastikan API URL benar di index.html

### Problem: Task tidak muncul setelah edit kategori
**Solution:**
- Refresh halaman
- Check browser console untuk errors
- Pastikan category_id di database sudah benar
