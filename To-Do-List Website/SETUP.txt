💾 PENYIMPANAN DATA:

Semua data disimpan di:
  → Browser Local Storage (persisten sampai clear cache)

Data yang disimpan:
  → Users (username & password)
  → To-Do List items
  → Session token
  → Activity logs


---

⚠️ CATATAN PENTING:

1. Data disimpan di Local Storage browser
2. Jika clear browser cache/data, semua data akan hilang
3. Untuk backup data, gunakan fitur export (jika tersedia)
4. Password disimpan dengan enkripsi sederhana (base64)
5. Untuk production, upgrade ke PHP/Node.js backend

---

❓ TROUBLESHOOTING:

Masalah: Halaman tidak muncul
Solusi: Pastikan file login.html dibuka di browser yang benar

Masalah: Data hilang setelah restart browser
Solusi: Ini normal. Local Storage terhapus jika clear cache.
        Gunakan backend server untuk persistent storage.

Masalah: Lupa password
Solusi: Clear browser cache dan register ulang

Masalah: Tidak bisa login
Solusi: Pastikan username & password benar
        Atau register akun baru
