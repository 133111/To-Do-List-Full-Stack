  💡 TIPS & TRIK
═══════════════════════════════════════════════════════════════

✓ Simpan session Anda:
  - Tekan Ctrl+Shift+S untuk save (Windows)
  
✓ Refresh halaman:
  - Tekan F5 jika data tidak muncul
  
✓ Lihat data di Local Storage:
  - Tekan F12 → Application → Local Storage
  
✓ Hapus semua data:
  - Buka DevTools (F12)
  - Console → localStorage.clear()
  - Refresh halaman
  
✓ Export data sebagai JSON:
  - Tekan F12 → Console
  - Ketik: localStorage
  - Copy semua data

═══════════════════════════════════════════════════════════════
  🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

MASALAH: Halaman putih / tidak muncul
SOLUSI:
  - Pastikan file login.html ada di folder
  - Gunakan browser terbaru (Chrome, Firefox, Edge)
  - Tekan F5 untuk refresh

MASALAH: Tidak bisa login
SOLUSI:
  - Cek username & password (case sensitive)
  - Clear browser cache (Ctrl+Shift+Delete)
  - Coba register akun baru

MASALAH: Lupa password
SOLUSI:
  - Tidak ada fitur "Lupa Password"
  - Register akun baru
  - Atau clear data & mulai ulang

MASALAH: Data hilang setelah tutup browser
SOLUSI:
  - Ini normal (Local Storage behavior)
  - Jangan clear browser cache
  - Untuk permanent storage, butuh server backend

MASALAH: Tugas hilang tiba-tiba
SOLUSI:
  - Tugas hilang otomatis setelah 2 jam
  - Catat waktu di log aktivitas sebelum hilang

MASALAH: Sidebar tidak bisa di-collapse
SOLUSI:
  - Klik tombol ☰ di atas
  - Di mobile, sidebar collapse otomatis

MASALAH: Chart tidak muncul di dashboard
SOLUSI:
  - Butuh ada minimal 1 log
  - Simpan log aktivitas dulu
  - Refresh halaman
